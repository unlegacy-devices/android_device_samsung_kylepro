Samsung Galaxy Galaxy Trend Lite GT-S7390
==============

### What does work
  - Audio
  - EGL
  - Wifi
  - Bluetooth
  - RIL (Partially)
  - External SD Card
  - Vibration
  - Buttons
  - Earjack
  - Camera (Partially)
  - MTP

### What doesn't work
  - Internal SD Card
  - Sensor
  - Microphone
  - And which is didn't test for now.
